ELSR Cambridge Cycling Route Database

We maintain a database of over 400 GPX files, of cycling routes based around Cambridge. These have all been ridden by us, and the routes have been created by our members to be cycling friendly, eg avoiding main roads and using scenic back roads, wherever possible etc.

The rides appear in the list below. Click on the name of a route to see the details, including the GPX file in Google Maps. All GPX files are free to download for signed up members of this site.

The list is maintained and updated by the community, so if you have a good route around Cambridge, please feel free to add it using the NEW ROUTE button below.
