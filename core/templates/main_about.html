I was bored, so I decided to learn Python. After a bit of googling I came across Udemy courses and picked their highest rated Python course: 100 Days of Code by Angela Yu. So, for a colossal outlay of £15.99, I started learning Python. The course was excellent and had a good chunk, maybe 20 days, focussed on designing and building websites using Flask / Jinja and the Bootstrap frameworks. After going through all this, I decided I needed a project to consolidate everything on, so I wrote this website from scratch.

So, what does this website use?

  • Python as the underlying back end language.
  • Flask / Jinja as the html templating engine.
  • CSS / Javascript styling from Bootstrap, this website is based on the Clean Blog theme we used in one of the course exercises.
  • Fancy font icons including the man on a bike from Font Awesome, the man on a bike comes from here.
  • Website fonts from Google Fonts.
  • SQL databases courtesy of SQLAlchemy.
  • Web forms are handled using Flask-WTF.
  • Password hashing using Werkzeug Utilities.
  • Google Map plugins using Google Maps API. I'll probably swap this out for OSM as it costs me cold hard cash everytime you see a map.
  • Email and web domain courtesy of Google Domains, a whole £10 to buy elsr.co.uk and £6 per month to run an email server!
  • GPX file parsing using gpxpy
  • Most of the background images are my own, although I might have stolen one or two from the internet.
  • Fancy user avatars, for logged in users, from Gravatar. Mine is "Blake et Mortimer" stolen from their facebook page.
  • This website uses Gunicorn as an application server and NGINX as a reverse proxy.
  • The site is hosted on a Virtual Machine in the Google Cloud in one of their London data centers.
  • Site diagnostics and real time performance data courtesy of Sentry.

The source code is on Github, you can take a look here.
